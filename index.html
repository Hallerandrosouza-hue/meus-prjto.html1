O que é cada tecnologia

HTML

Linguagem de marcação para estruturar conteúdo. Pense em seções, títulos, parágrafos e formulários.

CSS

Folhas de estilo para aparência. Define cores, fontes, espaçamentos e organização visual.

JavaScript

Linguagem de programação executada no navegador. Permite interatividade, validações e respostas a eventos.

PHP

Linguagem de programação executada no servidor. Recebe dados do formulário, processa e devolve uma resposta.

Formulário simples

O atributo method="post" envia os dados no corpo da requisição.
O atributo action indica o arquivo no servidor que fará o processamento.
Cada campo tem um atributo name, que será a chave usada pelo PHP para ler o valor enviado.
